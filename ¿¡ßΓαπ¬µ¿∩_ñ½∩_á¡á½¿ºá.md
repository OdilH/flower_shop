# ИНСТРУКЦИЯ ПО ВЫПОЛНЕНИЮ ЭТАПА 1

## Быстрый старт

1. Откройте сайт **samson-buket.ru** в браузере Chrome
2. Откройте Chrome DevTools (F12)
3. Следуйте инструкциям ниже для каждого раздела

---

## 1.1 СТРУКТУРНЫЙ АНАЛИЗ

### Как составить карту сайта:

1. **Откройте главную страницу** и пройдитесь по всем ссылкам в меню
2. **Запишите все найденные страницы** в раздел 1.1 файла `ЭТАП_1_АНАЛИЗ_САЙТА.md`
3. **Создайте скриншот навигации** (сохраните в папку `materials/navigation/`)

**Инструменты:**
- Используйте расширение браузера "Sitemap Generator" для автоматического создания карты сайта

---

## 1.2 ТЕХНИЧЕСКИЙ АНАЛИЗ

### Определение технологий:

1. **Установите расширение Wappalyzer:**
   - Chrome Web Store → поиск "Wappalyzer" → Установить
   - Откройте сайт → кликните на иконку Wappalyzer → увидите все технологии

2. **Проверка через DevTools:**
   - Откройте вкладку **Network** (F12 → Network)
   - Обновите страницу (F5)
   - Найдите файлы `.css`, `.js` - запишите их названия
   - Проверьте заголовки ответов сервера (Headers) для определения Backend

3. **Анализ HTML:**
   - Правый клик → "Просмотреть код страницы" (View Page Source)
   - Изучите структуру тегов и классов
   - Запишите основные классы и ID в раздел 1.2

4. **Анализ CSS:**
   - DevTools → вкладка **Sources** → найдите файлы `.css`
   - Изучите медиа-запросы (@media) - это breakpoints
   - Запишите найденные breakpoints

5. **Анализ JavaScript:**
   - DevTools → вкладка **Sources** → найдите файлы `.js`
   - Проверьте наличие библиотек (jQuery, Swiper и т.д.)
   - Запишите найденные библиотеки

---

## 1.3 ДИЗАЙН АНАЛИЗ

### Определение цветов:

1. **Через DevTools:**
   - Выберите элемент на странице (кнопка, заголовок)
   - DevTools → вкладка **Elements** → найдите элемент
   - В правой панели найдите `color` или `background-color`
   - Скопируйте hex-код цвета (например, #e91e63)

2. **Используйте инструмент "Пипетка":**
   - DevTools → вкладка **Elements** → выберите элемент
   - В панели стилей нажмите на цветной квадратик
   - Откроется палитра цветов с пипеткой

3. **Запишите все цвета** в раздел 1.3

### Определение шрифтов:

1. **Через DevTools:**
   - Выберите текстовый элемент
   - В панели стилей найдите `font-family`
   - Запишите название шрифта

2. **Проверка загруженных шрифтов:**
   - DevTools → вкладка **Network** → фильтр "Font"
   - Обновите страницу
   - Увидите все загруженные шрифты

### Измерение размеров:

1. **Используйте инструмент измерения:**
   - DevTools → вкладка **Elements**
   - Выберите элемент
   - В правой панели увидите размеры (width, height, padding, margin)

2. **Или используйте линейку:**
   - Установите расширение "Page Ruler" для точных измерений

### Создание скриншотов:

1. **Полноэкранные скриншоты:**
   - Установите расширение "Full Page Screen Capture"
   - Или используйте встроенный инструмент Chrome: F12 → Cmd/Ctrl + Shift + P → введите "screenshot"

2. **Скриншоты отдельных элементов:**
   - Выберите элемент → правый клик → "Capture node screenshot"

3. **Сохраните скриншоты** в папку `materials/screenshots/`

## 1.4 ФУНКЦИОНАЛЬНЫЙ АНАЛИЗ

### Тестирование форм:

1. **Откройте форму обратной связи**
2. **Попробуйте отправить пустую форму** - проверьте валидацию
3. **Заполните форму неправильными данными** - проверьте валидацию
4. **Заполните форму правильно** - проверьте отправку
5. **Запишите все поля и правила валидации**

### Тестирование корзины:

1. **Добавьте товар в корзину**
2. **Проверьте:**
   - Обновился ли счетчик корзины
   - Сохранился ли товар после обновления страницы (localStorage)
   - Можно ли изменить количество
   - Можно ли удалить товар
   - Правильно ли считается сумма

3. **Откройте DevTools → Application → Local Storage** - проверьте, что сохраняется

### Тестирование фильтров:

1. **Откройте каталог**
2. **Примените каждый фильтр**
3. **Проверьте, как изменяется список товаров**
4. **Запишите все типы фильтров**

### Тестирование модальных окон:

1. **Найдите все элементы, которые открывают модальные окна** (кнопки, ссылки)
2. **Кликните на каждый** - проверьте, что открывается
3. **Проверьте закрытие** (крестик, клик вне окна, ESC)
4. **Запишите все типы модальных окон**

### Тестирование слайдеров:

1. **Найдите все слайдеры на сайте**
2. **Проверьте:**
   - Автопрокрутку (если есть)
   - Навигацию (стрелки, точки)
   - Перетаскивание (swipe)
3. **Запишите настройки каждого слайдера**

---

## 1.5 АДАПТИВНОСТЬ

### Проверка на разных разрешениях:

1. **Откройте DevTools** (F12)
2. **Включите режим адаптивного дизайна:**
   - Нажмите Cmd/Ctrl + Shift + M
   - Или кликните на иконку устройства в панели DevTools

3. **Проверьте следующие разрешения:**
   - Desktop: 1920px, 1440px, 1200px
   - Tablet: 1024px, 768px
   - Mobile: 414px, 375px, 320px

4. **Для каждого разрешения:**
   - Сделайте скриншот
   - Запишите изменения в layout
   - Проверьте работу меню
   - Проверьте читаемость текста

5. **Определите breakpoints:**
   - Измените ширину окна
   - Найдите точки, где layout меняется
   - Запишите эти значения (например, 768px, 1024px)

---

## 1.6 СБОР КОНТЕНТА

### Скачивание изображений:

1. **Логотип:**
   - Правый клик на логотип → "Сохранить изображение как..."
   - Сохраните в `materials/images/logo/`

2. **Изображения товаров:**
   - Откройте каталог
   - Правый клик на каждое изображение → "Сохранить изображение как..."
   - Или используйте расширение "Image Downloader" для массового скачивания
   - Сохраните в `materials/images/products/`

3. **Иконки:**
   - Если иконки в формате SVG: правый клик → "Сохранить как..."
   - Если иконки из шрифта (Font Awesome): используйте DevTools для определения класса
   - Сохраните в `materials/images/icons/`

4. **Баннеры:**
   - Найдите все баннеры на сайте
   - Сохраните в `materials/images/banners/`

### Копирование текстов:

1. **Выделите текст на странице**
2. **Скопируйте (Ctrl+C)**
3. **Вставьте в соответствующий раздел файла `ЭТАП_1_АНАЛИЗ_САЙТА.md`**

**Или используйте расширение:**
- "Copy All URLs" для копирования всех ссылок
- "Text Grabber" для массового копирования текста

### Сохранение ссылок:

1. **Найдите все ссылки на странице:**
   - DevTools → Console → выполните: `Array.from(document.querySelectorAll('a')).map(a => a.href)`
   - Скопируйте результат

2. **Разделите на:**
   - Внутренние ссылки (на том же домене)
   - Внешние ссылки (на другие домены)
