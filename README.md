# 🌹 Цветикс - Сайт доставки цветов

Онлайн-магазин авторских цветочных композиций с доставкой по Москве.

## ✨ Особенности

- 🛍️ Полнофункциональный каталог с фильтрацией
- 🛒 Корзина и оформление заказа
- ❤️ Избранное
- 📱 Адаптивный дизайн
- 🎨 Современный UI/UX
- 🔐 Админ-панель
- 📧 Уведомления по email

## 🛠️ Технологии

**Frontend:**
- HTML5, CSS3
- JavaScript (ES6+)
- jQuery
- Swiper.js (слайдеры)
- AOS (анимации при скролле)

**Backend:**
- PHP 7.4+
- MySQL 5.7+
- RESTful API

## 🚀 Быстрый старт

### Вариант 1: XAMPP (Windows)

1. Установите [XAMPP](https://www.apachefriends.org/)
2. Скопируйте проект в `C:\xampp\htdocs\samson\`
3. Запустите Apache и MySQL в XAMPP
4. Создайте БД `samson_buket` в phpMyAdmin
5. Импортируйте `database/schema.sql`
6. Настройте `api/config.php`
7. Откройте `http://localhost/samson/`

### Вариант 2: Встроенный PHP сервер

```bash
cd d:\samson
php -S localhost:8000
```

Откройте `http://localhost:8000`

### Вариант 3: Linux (Apache)

```bash
sudo cp -r samson /var/www/html/
sudo mysql -u root -p < database/schema.sql
# Настройте api/config.php
sudo systemctl restart apache2
```

## ⚙️ Настройка

### База данных

1. Создайте БД: `samson_buket`
2. Импортируйте схему: `database/schema.sql`
3. Настройте подключение в `api/config.php`

### Конфигурация

Отредактируйте `api/config.php`:
```php
define('DB_HOST', 'localhost');
define('DB_USER', 'root');
define('DB_PASS', '');
define('DB_NAME', 'samson_buket');
define('ADMIN_EMAIL', 'admin@example.com');
```
