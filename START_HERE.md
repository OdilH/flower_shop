# 🎯 НАЧНИТЕ ЗДЕСЬ

## Шаги для запуска сайта Букеты & цветы

### ⚡ Самый быстрый способ (5 минут)

1. **Установите XAMPP** (если ещё не установлен)
   - Скачайте: https://www.apachefriends.org/
   - Установите в стандартную папку

2. **Скопируйте проект**
   ```
   C:\xampp\htdocs\samson\
   ```

3. **Запустите серверы**
   - Откройте XAMPP Control Panel
   - Нажмите "Start" для Apache и MySQL

4. **Создайте базу данных**
   - Откройте: http://localhost/phpmyadmin
   - Создайте БД: `samson_buket`
   - Импортируйте файл: `database/schema.sql`

5. **Настройте подключение**
   - Откройте: `api/config.php`
   - Укажите (обычно для XAMPP):
     ```php
     DB_USER = 'root'
     DB_PASS = ''  // Пустой пароль
     ```

6. **Откройте сайт**
   ```
   http://localhost/samson/
   ```

### ✅ Проверка работы

1. Главная страница открывается
2. Каталог загружается
3. API работает: http://localhost/samson/api/products.php
4. Нет ошибок в консоли браузера (F12)

### 🆘 Проблемы?

**Ошибка подключения к БД:**
- Проверьте, что MySQL запущен
- Проверьте настройки в `api/config.php`

**Страница не открывается:**
- Проверьте, что Apache запущен
- Проверьте путь к файлам

**API не работает:**
- Проверьте права доступа к папке `api/`
- Проверьте логи Apache
