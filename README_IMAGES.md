# Изображения для сайта Букеты & цветы

## Структура папок

```
assets/img/
├── logo/
│   ├── logo.svg          - Логотип компании (SVG)
│   └── logo.png          - Логотип (PNG fallback)
├── products/
│   ├── product-1.svg     - Букет из 25 красных роз
│   ├── product-2.svg     - Букет тюльпанов
│   ├── product-3.svg     - Свадебный букет
│   ├── product-4.svg     - Композиция в корзине
│   ├── product-5.svg     - Букет из пионов
│   ├── product-6.svg     - Эксклюзивный букет
│   └── placeholder.svg    - Универсальный placeholder
├── about/
│   ├── about-1.svg       - Фото мастерской
│   ├── team-1.svg        - Флорист Анна
│   ├── team-2.svg        - Флорист Михаил
│   └── team-3.svg        - Флорист Елена

│   ├── visa.svg          - Логотип Visa

```

## Использование

Все изображения созданы в формате SVG для:
- ✅ Масштабируемости без потери качества
- ✅ Малого размера файлов
- ✅ Быстрой загрузки
- ✅ Поддержки всех современных браузеров

## Замена на реальные изображения

Когда появятся реальные фотографии товаров:

1. **Товары** - замените файлы `product-*.svg` на JPG/PNG фотографии
2. **О команде** - замените `team-*.svg` на реальные фото сотрудников
3. **О компании** - замените `about-1.svg` на фото мастерской
4. **Логотип** - замените `logo.svg` на финальный логотип

### Форматы для замены:
- **Товары**: JPG, 800x800px, оптимизированные для web
- **Команда**: JPG, 400x400px, круглые аватары
- **Логотип**: SVG или PNG, прозрачный фон

## Fallback

В CSS добавлены градиентные фоны для случаев, когда изображения не загружаются:
- Товары: розовый градиент (#fce4ec → #f8bbd9)
- Команда: голубой градиент (#e3f2fd → #bbdefb)

## Оптимизация

Все SVG файлы оптимизированы и готовы к использованию. При замене на растровые изображения:

1. Используйте инструменты оптимизации (TinyPNG, ImageOptim)
2. Сохраняйте в формате WebP для современных браузеров
3. Добавьте fallback в формате JPG

## Примечания

- Все placeholder изображения имеют тематический дизайн
- Цветовая схема соответствует бренду (розовые оттенки)
- SVG файлы можно редактировать в любом векторном редакторе
